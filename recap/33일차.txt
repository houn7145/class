CSS  선택자
- 선택자란? HTML 문서 중 특정 부분을 선택하여 해당 부분의 속성을 변경하는 목적으로 사용 

태그선택자
tag 선택자는 특정 태그를 선택하여 CSS속성을 적용할 수 있음

전체(*) 선택자 ; 페이지 초기화
전체라는 의미의 * 선택자를 사용하면 문서 전체를 선택하는 의미가 있음 주로 페이지를 초기화할 때 사용

id(#)선택자 class(.)선택자
태그의 id와 class를 이용하여 CSS에 속성을 설정할 수 있음
id는 #으로 class는 .으로 사용

전체 선택자와 그 외 혼합
ag, id, class 혼합 : tag, id, class를 혼합하여 CSS에서 속성을 설정할 수 있음

속성 선택자
tag의 속성을 선택하여 CSS의 속성을 설정할 수 있음
선택자[속성] : 지정한 속성을 가지고 있는 요소를 찾아 스타일을 적용
선택자[속성=값] : 속성 값이 특정값과 같은 경우를 찾아 스타일을 적용
선택자[속성^=값] : 속성 값이 지정한 문자로 시작하는 속성값에 대해서만 스타일을 적용
선택자[속성$=값] : 속성 값이 지정한 문자로 끝나는 속성에 대해서만 스타일을 적용

후손선택자, 자손선택자(바로아랫단)
ag의 속성을 선택하여 CSS의 속성을 설정할 수 있음
ex) div p, div > p

동위(형제)선택자(+하나, ~다)
(+하나만, ~다선택) : 태그의 동등한 위치를 판단하여 CSS속성을 설정할 수 있음

반응선택자(마우스를 올렸을 때)
마우스를 올려 놓았을 때(hover상태)의 반응에 따른 속성을 설정할 수 있음

상태선택자
상태에 따라 CSS속성이 변화하는 설정을 할 수 있음  
ex) input태그의 경우 focus가 맞혀졌을 때

구조선택자
구조에 따라 CSS 속성이 변화하는 설정을 할 수 있음
ex) #nav ul li:nth-child(2n) : ul밑의 li중 짝수번째
