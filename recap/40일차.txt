함수 - 가변인자함수, 내부함수, 자기호출함수, 콜백함수, 클로저함수, 내장함수

가변인자함수 : 매개변수를 선언된 형태와 다르게 사용했을 때 매개변수를 모두 활용할 수 있고
매개변수의 갯수가 변할 수 있는 함수
ex) Array() 
arguments : 매개변수의 배열이며 시스템이 자동생성함

내부함수 : 함수내에 선언된 함수, 다른 함수와의 충돌을 피하고자 사용함
jQuery는 대부분 선언적 함수를 내부함수로 작성하고 있음

자기호출함수 : 다른 개발자가 만든 로직에 영향을 주지 않기 위해 함수를 생성하자마자 호출하는 함수
ex) (fuction(){ … })();

콜백함수 : 매개변수안에 있는 함수
자바스크립트에서는 함수도 하나의 자료형이므로 매개변수에 넣을 수 있으며 
매개변수에 넣어 전달된 함수를 콜백함수라 함

클로저함수 : return값이 함수면 클로저함수라 칭함

타이머함수 : 일정시간 후 또는 일정시간마다 반복적으로 실행할 로직을 만들때 사용함
setTimeout(function, millisecond) : 일정시간 후 함수를 한번 실행
setInterval(function, millisecond) : 일정시간마다 함수를 반복해서 실행
clearTimeout(id) : setTimeout 중지
clearInterval(id) : setInterval 중지

인코딩과 디코딩 함수 
인코딩함수 : 문자를 컴퓨터에 저장하거나 통신에 사용할 목적으로 부호화할때 사용하는 함수
웹에서는 통신할 때 한글 같은 유니코드 문자가 오작동을 일으킬 수 있으므로 인코딩해야 함
encodeURIComponent(uriConponent) ; 대부부의 문자를 모두 인코딩
디코딩함수 : 컴퓨터에 저장되어 있거나 통신을 완료한 뒤 인코딩된 문자를 다시 원래대로 돌리는 함수
decodeURIComponent(encodedURI);   대부분의 문자를 모두 디코딩

코드 실행함수
eval(수식이나 자바스크립코드) : 자바스크립 코드를 실행

숫자확인함수 
isFinite() ; number가 유한한 값인지 확인
isNaN() ; number가 NaN인지 확인

숫자변환함수
Number(), parseInt() - 숫자로 형변환
parseFloat() - 실수로 형변환
