기본내장객체

Array 객체
Array() : 빈 배열을 만듦
Array(number) : 매개변수만큼의 크기를 가지는 배열을 만듦
Array(mixed, ..., mixed) : 매개변수를 매열로 만듦
length : 배열 요소의 갯수를 알아냄
concat() : 매개변수로 입력한 배열의 요소를 모두 합쳐 매열로 만들어 return
join() : 배열안의 모든 요소를 문자열로 만들어 return
pop() : 배열의 마지막 요소를 제거하고 return
push() : 배열의 마지막 부분에 새로운 요소를 추가
reverse() : 배열 요소의 순서를 뒤집음
slice() : 배열 요소의 지정한 부분을 return
sort() : 배열 요소를 정렬
splice() : 배열 요소의 지정한 부분을 삭제하고 삭제한 요소를 return

Date 객체
toLocaleString() : 현재 날짜와 시간
toLocaleDateString() : 현재 날짜
toLocaleTimeString() : 현재 시간

브라우저 객체 모델

window 객체
URL : 열고자 하는 HTML 페이지의 URL
name : 윈도우 간 통신하는데 사용하는 윈도우 이름 또는 target 속성
features : 윈도우 출력 모양을 지정하는 옵션

window 형태 옵션
width : 새 창 넓이
height : 새 창 높이
location : 주소 입력창 유무
menubar : 메뉴의 유무
resizable : 화면 크기 조절 가능 여부
status : 상태 표시줄의 유무
toolbar : 상태 표시줄의 유무

open() 메소드
새로운 window 객체를 생성하는 메소드이며 단지 팝업창을 여는 것에서 끝나지 않고 윈도우 객체를 return

window 객체 기본 메소드
moveBy : 창의 위치를 상대적으로 이동
moveTo : 창의 위치를 절대적으로 이동
resizeBy : 창의 크기를 상대적으로 지정
resizeTo : 창의 크기를 절대적으로 지정
scrollBy : 창 스크롤의 위치를 상대적으로 이동
scrollTo : 창 스크롤의 위치를 절대적으로 이동
focus() : 창에 초점을 맞춤
blur() : 창 초점을 제거
close() : 창을 닫음 

screen 객체
웹 브라우저의 화면이 아니라 운영체제 화면이 속성을 가지는 객체

location 객체
브라우저의 주소 표시줄과 관련된 객체
