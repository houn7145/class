CRUD

React에서 페이지를 만들때 class를 이용하는 방법과 function을 이용하는 방법이 있는데
함수를 이용하는 방법이 대체로 많고 쉽지만 개념을 알기 위해선 class를 이용하여 컴포넌트화 시키는 방법도 알아야 함

한 문서에 모든 코드를 작성하면 코드가 너무 길어지고 가독성이 떨어지기 때문에 태그들을 컴포넌트화 시켜서 가독성을 높이는 것이 좋음

CRUD는 Create Read Update Delete의 약자이며 데이터를 만들고 읽고 수정하고 지우는 작업을 컴포넌트화 시켜서 해야함

props과 state의 차이점
props : 컴포넌트를 사용하는 외부자를 위한 데이터
state : 컴포넌트를 만드는 내부자를 위한 데이터

state의 값을 변경 하려면 setState() 함수를 이용

배열의 깊은 복사
1방법 [...] 이용
 var _contents = [...this.state.contents]; 
 _contents.push({id:this.max_content_id, title:_title, desc:_desc});

2방법 concat 이용
 var _contents = this.state.contents.concat(
{id:this.max_content_id, title:_title, desc:_desc}
);

3방법 Array.from 이용
var _contents = Array.from(this.state.contents);
_contents.push(
{id:this.max_content_id, title:_title, desc:_desc}
);

shouldComponentUpdate() 함수를 이용하면 데이터가 바뀌지 않을 시 불필요한 render를 막을 수 있음
