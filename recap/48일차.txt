리눅스
가상머신을 이용하여 os를 가상화
리눅스 설치 iso 이미지 파일을 다운로드후 가상머신(vmware, vb등)을 이용하여 설치
가상머신의 디스크 크기 설정은 넉넉히 잡아야 하며 설치하는 pc의 사양에 맞춰 설치를 해야함

리눅스는 파일을 효율적으로 관리하기 위해 디렉토리를 계층적으로 구성함 -> tree구조
모든 디렉토리의 출발점은 루트(root) 디렉토리이며 빗금(/)으로 표시

리눅스 명령어
pwd - 현재 디렉토리 확인
cd - 디렉토리 이동
ls - 디렉토리의 내용틀 출력
-a - 숨김 파일을 포함하여 모든 파일 목록을 출력
-d - 지정한 디렉토리 자체의 정보를 출력
-i - 첫 번째 행의  inode 번호를 출력
-l - 파일의 상세 정보를 출력
-A - 마침표와 마침표 두개를 제외한 모든 파일 목록을 출력
-F - 파일의 종류를 표시
-L -심벌릭 링크 파일의 경우 원본 파일의 정보를 출력
-R - 하위 디렉토리 목록까지 출력

리눅스 문서편집 - vi
:q - vi에서 작업한 것이 없을 때 그냥 종료
q! - 작업한 내용을 저장하지 않고 종료
:w - 작업한 내용을 저장
:wq, :wq! - 작업한 내용을 저장하고 종료

입력모드
i - 커서 앞에 입력
a - 커서 뒤에 입력
o - 커서가 위치한 행의 다음 행에 입력
I - 커서가 위치한 행의 첫 칼럼으로 이동
A - 커서가 위치한 행의 마지막 칼럼으로 이동
O - 커서가 위치한 행의 앞 행에 입력

환경변수 설정(.bashrc 파일에 아래 내용 추가)
– $ vi ~/.bashrc
export JAVA_HOME=~/jdk1.8.0_202
export PATH=$PATH:$JAVA_HOME/bin

환경변수 적용
– $ source ~/.bashrc
source 명령은 명령을 실행한 터미널에만 환경변수가 적용
– .bashrc 파일을 수정하면 source 명령을 다시 실행해야 함.
리눅스를 재시작 하면 모든 터미널에 적용됨
